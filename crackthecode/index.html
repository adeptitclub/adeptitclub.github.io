  1. Given an array of integers, create a function to sort the elements of the array in decreasing order of number of ones in binary representation of each element. If two numbers have equal number of 1s in their binary representation, then the number which appeared first in the original array must appear first in the sorted array. For example, if the original array contains 32, 54, 90, 87, 76, 21 The output will be 87, 54, 90, 76, 21, 32 Explanation: Each number and its binary representation is as below Number Binary Number of 1s 32 00100000 1 54 00110110 4 90 01011010 4 87 01010111 5 76 01001100 3 21 00010101 3 So, sorted array in decreasing order of number of 1s will be 87, 54, 90, 76, 21, 32
  2. Thor of Asgard has been assigned the task of finding the lost tesseract by his father Odin. He has travelled 8 realms but did not find the tesseract anywhere. Finally, he heads towards the 9th realm, Midgard (Earth). In Midgard, he seeks help of Dr. Strange in finding the tesseract. Dr. Strange gives him a list of locations where the tesseract possibly could be. However, the list looks a bit strange to Thor and he has trouble understanding the Midgardian address format. Your task is to convert the Midgardian addresses into Asgardian address format so that Thor can understand them. If you are able to help him find the tesseract, he might recommend you for a position in the team Avengers. Good Luck. Given a text file midgardian_addresses.txt which contains street addresses in the given format <street details>, <city name>, <zip code> Create a new file asgardian_addresses.txt which contains addresses in the format <city name> - <zip code> Conditions:
    • Each address in the file is in it own separate line.
    • Maximum length of an address is 255 characters.
    • Maximum length of city name is 20 characters.
    • Zip codes are always 6 digits.
    • The format of street details can vary.

    Example: If the input file contains following addresses: Wayne Manor, Gotham, 132468 203, Avengers Tower, New York, 324546 The result file will contain: Gotham - 132468 New York - 324546
